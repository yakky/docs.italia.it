Come pubblicare

A chi è dedicato

Su Docs Italia puoi pubblicare qualsiasi tipo di documento della Pubblica Amministrazione. Docs Italia mette inoltre a disposizione un team di technical writer per aiutare le Pubbliche Amministrazioni e favorire il processo di pubblicazione.

Contattaci su Slack o scrivici su Forum Italia

  • se lavori in una PA e vuoi saperne di più o iniziare a pubblicare qualche documento;
  • se lavori in qualità di fornitore della PA e, in accordo con il committente, vuoi provare a utilizzare Docs Italia per pubblicare documentazione tecnica;
  • se, come cittadino, vuoi suggerire che un documento pubblico venga pubblicato su Docs Italia.

I passi da seguire

Per pubblicare i tuoi documenti puoi seguire questi cinque passi:

  1. Scarica lo Starter Kit
  2. Crea un account su GitHub
  3. Abilita il tuo account
  4. Converti il tuo documento
  5. Pubblica il tuo documento

1. Scarica lo Starter Kit

All’interno della Guida a Docs Italia puoi trovare uno Starter Kit, che ti permette di iniziare più rapidamente a pubblicare i tuoi documenti su Docs Italia. Nel kit trovi:

  • Un esempio di repository di configurazione;
  • Un esempio di repository di un documento;
  • Le istruzioni per l’uso e la modifica dei file.

2. Crea un account su GitHub

La registrazione è gratuita e può essere completata seguendo le istruzioni nella pagina ufficiale di GitHub (in inglese).

Come ente pubblico puoi scegliere se usare un account personale oppure crearne uno a nome del tuo ente. Se è la prima volta che pubblichi su Docs Italia per il tuo ente, è anche necessario creare un'organizzazione dedicata su GitHub e un repository al suo interno che la identifica.

Per approfondire, fai sempre riferimento alla Guida a Docs Italia.

Se ti trovi in difficoltà, scrivici su Slack!

3. Abilita il tuo account

Dopo aver creato l'utente e l'organizzazione su GitHub e aver aggiunto il repository necessario, l'unico passo da fare è chiedere l'autorizzazione alla pubblicazione su Docs Italia.

Puoi farlo contattandoci tramite il canale Slack di Docs Italia (scopri di più nel capitolo dedicato nella Guida a Docs Italia).

4. Converti il tuo documento

I documenti su Docs Italia sono pubblicati in formato RST.

Puoi scrivere i documenti direttamente in RST, oppure se il tuo documento è già in un altro formato (es. ODT, DOCX, o Markdown) puoi usare questo convertitore:

5. Pubblica il tuo documento

A questo punto puoi iniziare a creare il tuo repository di documento usando lo starter kit (scopri di più nel capitolo dedicato nella Guida a Docs Italia), e pubblicare i tuoi documenti!
